# NPM Commands (Angular, Vue, React)

## Angular (CLI)
```
npm install -g @angular/cli
ng new my-app
cd my-app
ng serve
```
Explanation:
- Install the Angular CLI globally
- Create a new app
- Run the dev server

## Vue (create-vue)
```
npm create vue@latest
cd my-vue-app
npm install
npm run dev
```
Explanation:
- Create a Vue app with the official scaffolder
- Install dependencies
- Start the dev server

## React (Vite)
```
npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm run dev
```
Explanation:
- Use Vite to scaffold a React app
- Install dependencies
- Start the dev server

## Tailwind CSS (Vite)
```
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
Explanation:
- Install Tailwind and PostCSS tooling
- Generate Tailwind and PostCSS config files

## Bootstrap
```
npm install bootstrap
```
Explanation:
- Install Bootstrap CSS/JS in your project
